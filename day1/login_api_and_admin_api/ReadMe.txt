管理接口：
作用：
1、解锁用户
2、添加用户
3、删除用户


1、运行程序
python admin.py

2、提示功能，根据功能输入1-3的数字提示

解锁用户：
===============================================================
root@tim:/tim# python admin.py 
欢迎登录，登录流程控制台
输入1：解锁用户
输入2：添加用户
输入3：删除用户

请输入功能（1-3）：1
目前已锁账户为
-------------------------------------------
tianshuai
-------------------------------------------
请输入用户名：tianshuai   #这里输入上面显示被锁的用户
用户tianshuai已解锁，请重新登录！
root@tim:/tim# 
===============================================================
