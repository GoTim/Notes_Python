一、程序介绍

此程序是一个3级菜单！

具体树状结构为：

city_map = {"河北省":{"石家庄市":['平山县','无极县','灵寿县'],
                    "衡水市":['饶阳县','安平县','武强县']
                    },
            "山东省":{"济南市":['济阳县','商河县','平阴县'],
                    "青岛市":['市南区','市北区','市东区'],
                   },
            "河南省":{"洛阳市":['新安县','洛宁县','宜阳县'],
                    "郑州市":['金水区','惠济区','上街区']
                   }
}

二、程序
1、运行程序：
python map.py

2、使用介绍
第一层菜单：
---------------------地图---------------------------
山东省
河北省
河南省
----------------------------------------------------

请输如您要查看的省名: 河北省
-----------------河北省包含的市为--------------------

衡水市
----------------------------------------------------

第三层菜单：
请输入你要查看的市名：衡水市
-----------------衡水市包含的县为--------------------
饶阳县
安平县
武强县
-----------------------------------------------------
请问是否退出：1 退出、2 返回最上层,输入任意键返回上一层：


#最后一层菜单会根据您的输入返回和跳转：
